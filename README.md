# Molebatso — Portfolio (starter)

This is a simple, responsive single-page portfolio template built with plain HTML, CSS, and a tiny JavaScript file.

## Files
- `index.html` — main page (replace name, bio, project details).
- `styles.css` — all styling (variables at top for quick theme edits).
- `script.js` — smooth scrolling, project modal, contact helper.
- `README.md` — this file.

## Quick customization
1. Open `index.html` and replace:
   - The name "Molebatso" (title, header, footer).
   - The placeholder hero text and the About section.
   - Project cards: change `data-title`, `data-desc`, `data-image`, and `data-link` for each `.project-card`.
   - Replace placeholder images with your images stored in an `assets/` folder (create one).
   - Update the contact email in both `index.html` and `script.js` (search for `hello@example.com`).

2. Colors & fonts
   - Edit CSS variables at the top of `styles.css` (accent color, background, etc).
   - Swap the Google Font in `index.html` if you prefer another font.

3. Add resume
   - Drop `resume.pdf` into the repo and link it from the header or About section.

## Deploy to GitHub Pages
1. Create a new repository named `molebatso.github.io` (replace with your username).
2. Push these files to the `main` branch of that repo.
3. Visit `https://molebatso.github.io` after a minute. GitHub Pages will serve the static site.

Alternatively, deploy to Netlify, Vercel, or any static host by pointing to this folder.
